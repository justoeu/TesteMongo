# Glossário

| Termo | Definição |
|-------|-----------|
| **Pace** | Tempo necessário para percorrer 1 km. Expresso como `M:SS/km` |
| **Velocidade média** | Distância sobre tempo total. Expressa em `km/h` |
| **Split** | Tempo parcial em um ponto/distância intermediária da corrida |
| **5 km** | Distância clássica de prova de rua |
| **10 km** | Distância clássica de prova de rua |
| **Meia maratona** | 21,0975 km (21.097,5 m) — distância oficial IAAF/World Athletics |
| **Maratona** | 42,195 km (42.195 m) — distância oficial IAAF/World Athletics |
| **Tempo alvo** | Meta de tempo total que o corredor deseja atingir |
| **Z2 / Z3** | Zonas de treino aeróbico (não usadas no MVP) |
| **Negative split** | Estratégia de correr a 2ª metade mais rápida que a 1ª (não calculada no MVP) |
| **HALF_UP** | Modo de arredondamento padrão: 0,5 arredonda para cima |
